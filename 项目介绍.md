# CodeCli 服务介绍

## 什么是 CodeCli？

CodeCli 是专业的 Claude Code API 中转服务平台，让开发者能够以合理的成本使用 Claude Code AI 编程助手，提升开发效率。

## 服务特点

### 独享服务
- **非拼车模式**：独立 API 服务，不与他人共享
- **隐私安全**：代码和对话内容完全隔离
- **稳定可靠**：不受其他用户影响

### 稳定高速
- **住宅 IP**：采用稳定高速住宅 IP，避免风控
- **低延迟**：优化网络链路，响应迅速
- **高可用**：分布式架构，服务稳定

### 灵活计费
- **按量计费**：用多少付多少，无最低消费，Token 消耗透明可查
- **日卡套餐**：适合临时项目或试用，购买后立即生效
- **月卡套餐**：长期使用更划算，整月无忧使用

## 适用人群

| 套餐 | 适合人群 |
|------|----------|
| 按量计费 | 使用频率不固定的开发者 |
| 日卡 | 短期项目、临时需求、试用体验 |
