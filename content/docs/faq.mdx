本文档收集了使用 CodeCli 服务过程中的常见问题及解答。

## 关于服务

### Q: CodeCli 提供什么服务？

**A:** CodeCli 提供 Claude Code API 中转服务，支持三种计费方式：
- **按量计费**：用多少付多少
- **日卡套餐**：适合短期使用
- **月卡套餐**：长期使用更划算

### Q: 这是拼车服务吗？

**A:** 不是。我们提供独立的 API 服务，非拼车模式：
- 独享服务资源，不与他人共享
- 代码和对话内容完全隔离
- 不受其他用户影响，服务稳定

### Q: 服务稳定吗？

**A:** 是的，我们采用：
- 稳定高速住宅 IP，避免风控
- 分布式架构，支持故障自动切换
- 7x24 小时监控，异常及时处理

## 计费相关

### Q: 如何选择计费方式？

**A:** 根据您的使用情况选择：

| 计费方式 | 适合场景 |
|---------|---------|
| 按量计费 | 使用频率不固定，偶尔使用 |
| 日卡 | 短期项目、临时需求、试用体验 |
| 月卡 | 日常开发、长期项目、高频使用 |

### Q: 如何查看使用量？

**A:** 我们提供用量查询功能，您可以实时查看 Token 消耗情况和账单明细。

### Q: 支持哪些支付方式？

**A:** 支持主流支付方式，具体请联系客服咨询。

## 使用问题

### Q: 如何开始使用？

**A:**
1. 联系客服购买服务
2. 获取 API Key
3. 配置 Claude Code 使用我们的 API 端点
4. 开始使用

### Q: 支持哪些 Claude 模型？

**A:** 支持 Claude Code 官方支持的所有模型，包括最新的 Claude 4 系列。

### Q: 响应速度如何？

**A:** 我们采用稳定高速住宅 IP 和优化的网络链路，保持低延迟响应。

### Q: 支持哪些编程语言？

**A:** Claude Code 支持主流编程语言：
- JavaScript / TypeScript
- Python
- Java / Go / Rust
- C / C++
- 以及更多...

## 安全与合规

### Q: 安全吗？

**A:** 是的，我们确保：
- 独立服务，数据隔离
- 不记录您的代码内容
- 合法合规运营

### Q: 使用有什么限制？

**A:** 请遵守以下规范：
- 以学习 AI 编程为主要目的
- 不得用于任何违规用途
- 遵守 Anthropic 使用条款

## 故障排除

### Q: 遇到连接问题怎么办？

**A:** 请检查：
1. API Key 是否正确配置
2. 网络连接是否正常
3. API 端点地址是否正确

如问题持续，请联系客服协助排查。

### Q: 响应速度变慢怎么办？

**A:** 可能的原因和解决方案：
1. 网络波动 - 稍后重试
2. 请求内容过长 - 减少单次请求代码量
3. 服务繁忙 - 错峰使用
